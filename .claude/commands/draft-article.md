# /draft-article — 기사 초안 생성 에이전트

보도자료, 취재 메모, 외신 번역 등 원본 자료로부터 울산매일UTV 스타일의 기사 초안을 생성합니다.

## 사용법

```
/draft-article
```

실행 후 다음 정보를 제공하세요:
1. **원본 자료**: 보도자료 전문 또는 취재 메모 붙여넣기
2. **기사 유형**: 스트레이트(사실 보도) / 피처(기획) / 해설 / 인터뷰 요약
3. **목표 카테고리**: 사회 / 문화 / 스포츠 / 인문학 / 스타트업·비즈니스
4. **목표 분량**: 단신(300자) / 보통(600자) / 심층(1000자+)

## 초안 생성 절차

### Step 1: 원본 분석
- 핵심 사실 추출 (5W1H: 누가, 언제, 어디서, 무엇을, 왜, 어떻게)
- 출처 명확성 확인
- 인용문 식별 및 분리

### Step 2: 보도자료 유사도 경고
- 원본 대비 유사도 70% 이상 시: "⚠️ 추가 독자 취재 권고"
- 50-70%: "✅ 적절한 재작성"
- 50% 미만: "✅ 높은 독창성"

### Step 3: 초안 생성 구조
```
[리드] (첫 문단 — 핵심 사실 압축, 50자 이내)
[본문] (역피라미드 구조 — 중요도 순)
[인용] ("..." — 출처 표기)
[배경] (독자 이해를 위한 맥락)
[마무리] (향후 전망 또는 관련 정보)
```

### Step 4: 메타데이터 제안
```
제목 (SEO): 35자 이내, 핵심 키워드 포함
부제목: 제목 보완, 독자 호기심 유발
카테고리: [자동 분류]
태그: ['키워드1', '키워드2', ...]
메타 설명: 150자 이내, 검색 결과 노출용
예상 읽기 시간: N분
```

## 스타일 가이드

**헤드라인 작성 원칙**
- 동사 포함 (명사형 나열 지양)
- 구체적 숫자, 고유명사 활용
- 클릭베이트 지양, 사실에 근거

**본문 스타일**
- 문장 길이: 40자 이하 권장 (모바일 가독성)
- 단락: 3-4문장, 빈 줄로 구분
- 수동태 지양, 능동태 사용
- 존댓말 불필요 (기사체)

**금지 표현**
- "~로 알려졌다" (미확인 정보)
- "~라고 한다" (출처 불명)
- "충격", "경악" 등 과장 표현
- 범죄 피의자 단정 표현 (무죄 추정 원칙)

## 주의사항
- AI 초안은 **반드시 기자가 직접 사실 확인** 후 수정하여 사용 (AI 가이드라인 제1조)
- 생성된 초안을 무수정 발행 금지 (윤리강령 위반)
- 외신 번역 시 저작권 표기 유지 (제6조)
- 취재원 개인정보가 담긴 메모는 AI 서비스에 입력 금지 (AI 가이드라인 제5조)
